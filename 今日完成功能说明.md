# AI知识系统 - 今日完成功能说明

## 项目概述
AI知识系统是一个基于Spring Boot + Vue.js + Python的智能问答系统，具备知识库管理、智能问答、对话上下文记忆等功能。

## 今日完成功能

- **前端连接已实现**：`index.js` 中 `sendMessageStream` 方法已实现SSE连接
- **安全认证已配置**：JWT认证和Spring Security配置支持SSE端点

### 2. Redis多级缓存优化 ✅

#### 缓存架构
```
┌─────────────────────────────────────────┐
│           多级缓存架构                   │
├─────────────────────────────────────────┤
│ 第一层：Caffeine本地缓存 (毫秒级响应)    │
│  - 问答结果缓存 (TTL=10分钟, max=1000)  │
│  - 用户会话缓存 (TTL=5分钟, max=500)    │
│  - 文档元数据缓存 (TTL=2分钟, max=200)  │
│  - 对话上下文缓存 (TTL=3分钟, max=300)  │
├─────────────────────────────────────────┤
│ 第二层：Redis分布式缓存 (秒级响应)       │
│  - 问答结果缓存 (TTL=1小时, max=10000)  │
│  - 用户会话缓存 (TTL=24小时, max=5000)  │
│  - 文档元数据缓存 (TTL=5分钟, max=1000) │
│  - 对话上下文缓存 (TTL=30分钟, max=2000)│
├─────────────────────────────────────────┤
│ 第三层：MySQL数据库 (持久化存储)        │
└─────────────────────────────────────────┘
```

#### 核心实现
- `MultiLevelCacheServiceImpl.java`：多级缓存服务实现
- `CacheConfig.java`：缓存配置和常量定义
- `CacheService.java`：缓存服务接口

#### 缓存策略
- **缓存穿透防护**：互斥锁防止重复查询数据库
- **缓存雪崩防护**：随机过期时间避免同时失效
- **热点数据**：永不过期标识，避免频繁刷新
- **缓存统计**：实时监控缓存命中率和性能

### 3. 对话上下文记忆功能 ✅

#### 记忆架构
```
┌─────────────────────────────────────────┐
│           对话记忆系统                   │
├─────────────────────────────────────────┤
│ 短期记忆 (Redis缓存)                    │
│  - 存储最近10轮对话                     │
│  - 快速访问，支持上下文关联             │
│  - TTL=30分钟，自动清理                 │
├─────────────────────────────────────────┤
│ 长期记忆 (MySQL数据库)                  │
│  - 存储对话摘要和关键信息               │
│  - 支持相似对话检索                     │
│  - 向量化存储，语义搜索                 │
├─────────────────────────────────────────┤
│ 记忆管理策略                            │
│  - 滑动窗口：保持最近N条消息            │
│  - 摘要压缩：长对话自动生成摘要         │
│  - 重要性加权：重要消息优先保留         │
└─────────────────────────────────────────┘
```

#### 核心实现
- `ConversationContextServiceImpl.java`：对话上下文服务
- `ConversationContext.java`：对话上下文实体
- `Message.java`：消息实体

#### 功能特性
- **上下文窗口管理**：动态调整对话历史长度
- **摘要生成**：自动提取对话核心内容
- **相似对话检索**：基于向量相似度的历史对话查找
- **重要性评分**：智能评估消息重要性，优化记忆保留

## 技术栈

### 后端技术栈
- **框架**：Spring Boot 3.x + MyBatis Plus
- **安全**：Spring Security + JWT
- **缓存**：Redis + Caffeine (多级缓存)
- **数据库**：MySQL 8.0
- **异步处理**：Spring @Async + ExecutorService
- **API文档**：Swagger/OpenAPI (待实现)
- **监控**：Spring Boot Actuator (待实现)

### 前端技术栈
- **框架**：Vue.js 3 + Element Plus
- **HTTP客户端**：Axios
- **状态管理**：Vuex/Pinia (待确认)
- **SSE处理**：原生ReadableStream API
- **UI组件**：Element Plus组件库

- **文档处理**：未集成 (PDF/Word解析)
